# LearnCpp.com

I am trying to transition from a python dev to a cpp dev and found this resource https://learncpp.com/#google_vignette that teaches cpp incrementally.

My biggest concern is writing canonical code that follows modern practices. This resource helps by:
- following modern standard (cpp17, 20, 23)
- warns of outdated c practices
- features best practice boxes that follow industry-standard approach

other resources to note:
- https://usaco.guide/ - USACO guide provides a comprehensive & organized roadmap to join USACO (pre-college competitive programming contests)
    - structured curriculum
    - problem sets
    - forum (USACO forum)

    * created by the CP initiative
    * start with the bronze section for basic stuff
