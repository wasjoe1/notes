# STL (standard template library) containers

```cpp
#include<vector> // tells the compiler where to find the defn of std::vector
using namespace std; // brings all names inside teh std:: namespace into the current scope => not recco-ed!!
vector<int>v; // creates the vector object, have to include <int> the template argument else it wont compile
```

STL containers include:
1. vectors          => == python's lists; dynamic arr
2. unordered_set    => == python's set
3. unordered_map    => == python;s dictionary
...
